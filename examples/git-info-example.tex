\bigskip

Defining a new \verb+dt+ format: \verb+\gitNewDateFormat+. Eg, the standard
format \verb+datetime+ is defined using: 
\begin{verbatim}
\gitNewDateFormat{datetime}{#1-#2-#3T#4:#5:#6#7#8:#9}
\end{verbatim}
in \texttt{git-info.sty}.
